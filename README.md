# Dungeon-Explorers

</div>

### 개요

Unity 6를 활용하여 **2D 플랫포머 모바일 디펜스 게임**을 제작하는 것이 목표입니다.

<br></br>

### 프로젝트 요약

| 정보 | 설명 |
|-----|-------|
| 게임 명 | 던전 탐험대(Dungeon Explorers) |
| 게임 장르| 2D, 디펜스, 액션, 캐주얼 |
| 사용 유니티 버전| Unity 6(6000.0.26f1) |
| 프로젝트 기간 | 2025.05.13 ~ 2025.05.21 |
